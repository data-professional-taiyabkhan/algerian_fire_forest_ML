Algerian Forest Fire Prediction

Using Machine Learning to Prevent Forest Fires

About the Dataset

This project is built on the Algerian Forest Fire dataset. The dataset includes information on various meteorological and vegetation-related factors collected across two regions in Algeria.

  • Variables: Temperature, Rain, RH (Relative Humidity), Wind Speed, and FFMC (Fine Fuel Moisture Code).
  • Target: The model predicts whether a forest fire will occur based on these variables.

About the Model

The machine learning model utilizes these variables to provide accurate predictions, helping in proactive fire prevention measures. Techniques like Random Forests and Logistic Regression are employed.
